# Backup
## 목적
- `Local Respository` : 지역저장소
- `Remote Resposity`: 원격저장소
-  `Local` >---`push`---> `Remote` > --- `pull` --- > `Local2`
    - `push`: 저장
    - `pull`: 불러오기

> 원격저장소를 통해 여러 개의 지역저장소에서 `push` 와 `pull`하여 협업이 가능하도록 함

## backup 방법
- `git remote -v`: 연결된 저장소 확인
- `git remote add origin [주소]` : 원격저장소 연결
-`git push`: 저장, 백업
- `git pull`: 가져오기

## 복제
- `git clone[주소][저장할 폴더]`
## 소스 코드
> 다양한 오픈소스 활용하기
